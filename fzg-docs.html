🔄 外包資料同步規則

排程

每日 09:05（台北時間），由 FZG agent 執行

資料來源（優先順序）

  1. 本機 Excel（SharePoint 同步）：F:\International Games System\研一外包_唯晶科技 - 桑比槍台\唯晶_桑比槍台_美術_工作日報.xlsx，分頁格式 YYYYMM
  2. Google Sheet（fallback）：唯晶日報 Sheet，分頁格式 2026/05
  3. 兩者都失敗 → 通知老爺

解析規則

  • NAME：xxx → 提取負責人
  • 數字開頭行（1. xxx）→ 提取工作項目
  • 含「請假」→ 建立一筆 task="請假", status="已完成"
  • 過濾任務名稱 ≤ 2 字元的項目
  • Pipeline 結果用 @(...) 強制轉陣列

寫入方式

  • 先 clear 當月分頁
  • 逐筆 add（每筆間隔 300ms，每 10 筆 sleep 1s）
  • 完成後 gviz 驗證筆數
  • 不使用 batch（已確認無效）

⚙️ 功能規範

看板功能

  • 三欄佈局：待辦 / 進行中 / 已完成
  • 負責人分組（可拖曳排序、收合/展開）
  • 任務卡：拖曳換欄、編輯、刪除、+子任務
  • 空欄自動收合，拖曳時自動展開
  • 待修正欄：無負責人的任務獨立顯示在待辦欄上方
  • 日期預設藍色，快到期/逾期變色（黃→橙→紅）
  • 不同月份拖曳操作不互相干擾

新增任務

  • 必填欄位：任務名稱、負責人（label 後加紅色「(必填)」）
  • 未填送出 → 紅框標示，填寫後消除
  • 取消時重置所有欄位狀態

待修正欄

  • 獨立欄區塊，在待辦欄上方
  • 紅色邊框，可收合/展開
  • 任務卡含 ✏️ 編輯 + 🗑️ 刪除按鈕
  • 刪除立即生效（fire-and-forget）
  • 無問題任務時不顯示
  • 時間軸不顯示這些任務

外包看板

  • 三區分組（一區/二區/三區）
  • 群組排序（▲▼），交換時兩個群組都閃爍
  • 同步按鈕：逐筆 add + 進度條顯示
  • 顏色說明與搜尋框同列

跨平台同步

  • 所有操作即時寫入 Sheet（需解鎖狀態）
  • 收合狀態、排序、群組名稱 → Sheet notes 分頁
  • 載入時從 Sheet 讀取覆蓋 localStorage
  • 點擊標題 → confirm → 清除快取 → spinner → 重新載入

📏 工作流程規範

  2. 整理需求與執行計畫
  3. 在頻道回覆確認
  4. 老爺確認後才開始實作

溝通規則

  • 老爺在哪個頻道發話，回覆就在同一頻道
  • 稱呼用戶為「老爺」
  • 使用中文
  • FZG 相關在 FZG 頻道（topic 2080）回覆

⚠️ 已知限制與注意事項

  • Apps Script 的 batch action 無效，必須用逐筆 add
  • PowerShell 的 ConvertTo-Json 必須加 -Depth 5 + UTF8.GetBytes，不加 -Compress
  • PowerShell pipeline 單一結果會變 String，需用 @(...) 強制轉陣列
  • no-cors fetch 回傳 opaque response，不能 await 判斷成功
  • gviz 讀取有快取延遲，驗證時可能需等幾秒
