三、資料同步機制

3.1 每日自動同步（09:05 執行）

資料流：

本機 Excel (F:\...\唯晶_桑比槍台_美術_工作日報.xlsx)
    ↓ PowerShell (sync_outsource.ps1)
    ├→ 外包 Sheet（追蹤器讀取用）
    └→ 唯晶日報 Google Sheet（備份）

流程：

  1. 優先用 Excel COM 讀取本機 Excel（分頁名：YYYYMM 如 202605）
  2. Excel 鎖住時 fallback 到唯晶日報 Google Sheet（gviz）
  3. 解析：NAME 行取負責人，數字開頭行取任務，「請假」特殊處理
  4. 過濾：任務名稱 ≤2 字元（除「請假」）排除
  5. 用 @() 強制陣列（防止單筆時 PowerShell 回傳 String）
  6. clear 外包 Sheet 當月分頁
  7. 逐筆 add（每筆間隔 300ms）
  8. gviz 驗證筆數
  9. 同步到唯晶日報（raw rows）
  10. 存同步時間到主 Sheet notes

3.2 前端即時同步（設定/狀態）

方式：

  • 小資料（URL < 2000 字元）→ GET SCRIPT_URL?action=saveNote&month=KEY&text=VALUE
  • 大資料（URL ≥ 2000 字元）→ navigator.sendBeacon POST JSON

觸發時機：每次用戶操作（分組、改名、排序、開合等）立即觸發

存儲位置：

  1. localStorage（本機即時生效）
  2. Sheet notes 分頁（雲端同步）

讀取時機：頁面載入時 loadNotes() 從 Sheet notes（headers=0）讀取 → 寫入 localStorage → 渲染

3.3 前端即時同步（任務資料）

方式：POST Content-Type: text/plain（simple request，不觸發 CORS preflight）

操作：update、delete、add、updateSort

確認方式：.then() 回調顯示同步狀態

五、前端功能架構

5.1 內部頁面

  • 看板：三欄（待辦/進行中/已完成）+ 待修正欄（無負責人任務）
  • 時間軸：Gantt 圖，按負責人分組，子任務縮排顯示
  • 報告：統計報表
  • 功能：拖曳換狀態、負責人排序/換區、任務排序、開合、搜尋、標籤篩選
  • 空欄自動收合：無任務的欄位預設收合，拖曳時展開

5.2 外包頁面

  • 看板：三區（一區/二區/三區），按負責人分組
  • 時間軸：Gantt 圖，按負責人分組，群組內任務 └ 前綴
  • 功能：手動分組（拖曳合併）、群組改名、▲▼ 排序、負責人換區（drop zone）、開合、搜尋
  • 分組模式：純手動（無 isSimilar 自動分組）
  • 群組名稱：首次建立時自動鎖定，之後只能手動 ✏️ 改名
  • 排序閃爍：交換位置時兩個元素都有 flash animation

5.3 共用功能

  • 月份切換（◀▶ + input type="month" 選擇器）
  • 管理模式（密碼：fzg）
  • 標題按鈕：confirm → 清快取 + 重載（從雲端讀取最新設定）
  • 🔄 重新載入：confirm → 從 Sheet 重新讀取顯示（不寫入）
  • 同步狀態指示器（🔄 同步中 / ✅ 已同步 / 📤 已送出 / ❌ 同步失敗）
  • 同步時間顯示（🕐 更新：日期時間）

七、禁止事項

  1. 前端「🔄 重新載入」按鈕不做任何寫入/清除操作
  2. 不使用 mode:'no-cors'（會導致設定 body 無法正確送達）
  3. 每日自動同步是唯一的外包資料寫入來源（前端不寫入外包資料）
  4. 不使用 isSimilar 自動分組（已移除）
  5. 不主動標記任務為 __independent__（移出群組用 delete 而非標記）

八、密碼與權限

  • 管理密碼：fzg
  • 解鎖後顯示：.edit-ctrl 元素（編輯、新增、排序、拖曳、刪除）
  • 鎖定時：唯讀模式，所有編輯控制隱藏
  • 外包「🔄 重新載入」不需解鎖

十、任務卡唯一 ID

外包任務卡使用 負責人_工作項目_開始日 作為唯一識別：

  • 例：Steven 林建憲_UI 視覺設計_2026/5/4
  • 用於：_manualBoardGroups key、data-board-item 屬性、排序陣列
  • 顯示時仍用任務名稱（t['工作項目']）
