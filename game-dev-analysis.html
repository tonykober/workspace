核心判斷

團隊技術能力完全足夠。30 隻現有角色完全覆蓋新遊戲 8 種體型需求，不需要新建角色模型或骨架。挑戰在於「風格轉換」— 透過改貼圖將恐怖風格轉為歡樂風格。

製作方向三要點

  1. 圓潤化 — 所有造型去掉尖角、複雜配件，強調球形/膠囊形基底
  2. 提亮化 — 背景冷色低飽和，角色暖色高飽和，特效最高飽和
  3. 減法設計 — 55 吋螢幕 1~2 公尺觀看距離，細節看不到，輪廓和色塊才是重點

風格參考標竿

  • Fall Guys 的圓潤感
  • Brawl Stars 的色彩飽和度
  • Splatoon 的街頭潮流感
  • 目標：讓 6 歲覺得好玩，12 歲覺得不丟臉

需要的心態轉換

團隊習慣做「細節豐富」的角色，新專案要學會「少即是多」：

  • 減少配件裝飾 → 強化輪廓辨識
  • 減少材質細節 → 強化色塊對比
  • 減少暗色調 → 強化明亮歡樂感

色彩規範

  • 背景層：冷色調 / 飽和度 30~50% / 明度 40~60%
  • 角色層：暖色調 / 飽和度 70~90% / 明度 60~80%
  • 特效層：任意色相 / 飽和度 90~100% / 明度 80~100%

角色視覺規範

  • 剪影測試：每個角色純黑剪影必須一眼可辨
  • 主色分配：8 角色各有專屬主色（紅/藍/綠/黃/橙/紫/粉/白）
  • 體型差異：矮胖/高瘦/方正/不定形/長腿/小重/大輕/帶刺
  • 表情系統：大眼睛 + 3~4 種表情切換（開心/驚嚇/生氣/暈眩）
  • Squash & Stretch：所有動作加入擠壓拉伸變形
  • 面數預算：3,000~5,000 面/角色
  • 貼圖：1024×1024，手繪色塊+漸層

場景視覺規範

  • 可讀性優先：地面/牆壁/障礙物色彩分明，不混淆
  • 機關高亮：可互動物件用高飽和色+微發光標示
  • 安全區標示：場地邊緣用明確視覺提示（虛線/漸變）
  • 動態元素：傳送帶、旋轉台用動畫+粒子強調運動方向
  • 面數預算：單地圖 50,000~80,000 面

特效視覺規範

  • 打擊回饋：速度線 + 星星 + 震動 + 短暫慢動作
  • 得分回饋：數字彈出 + 光圈擴散 + 音效同步
  • 連擊系統：combo 數字越大，特效越誇張（顏色升級、粒子增多）
  • 淘汰特效：被推出場外的誇張飛行軌跡 + 爆炸星星

UI 視覺規範

  • 圓角矩形為基礎形狀
  • 高對比色（深底+亮字 或 亮底+深字）
  • 字體大、粗，1~2 公尺外可讀
  • 動態過場（滑入/彈出/縮放）

場景 NPC 設計（敵對/中立）

NPC 1 — 搗蛋鬼（Trickster）

  • 定位：場景干擾者
  • 出現：收集狂熱、障礙賽跑
  • 行為：撒金幣/擋路/被撞誇張跌倒
  • 視覺：比玩家小一號，螢光色+搗蛋面具
  • 動畫：跑步循環、撒東西、被撞跌倒（3 組）

NPC 2 — 砲彈手（Bomber）

  • 定位：遠程威脅製造者
  • 出現：躲避球風暴、山丘之王
  • 行為：站高台發射砲彈/定時丟炸彈清場
  • 視覺：比玩家大一號，工程背心+安全帽
  • 動畫：待機搖晃、瞄準蓄力、發射（3 組）

NPC 3 — 裁判（Referee）

  • 定位：規則提示+氣氛製造
  • 出現：所有模式通用
  • 行為：開場吹哨/淘汰時誇張反應/待機搞笑互動
  • 視覺：黑白條紋衣+哨子
  • 動畫：吹哨、揮旗、誇張驚嚇（3 組）

跟班 NPC 設計（協力）

  • 觸發方式：玩家撿到「召喚道具」（召喚鈴鐺/友誼球）
  • 持續時間：10~15 秒
  • 跟班行為：
    • 自動追蹤最近敵方玩家並推擊
    • 或幫擋一次攻擊後消失
    • 或幫搶旗/金幣後交給玩家
    • 時間到自動「彈飛離場」（搞笑退場）
  • 跟班池：5~8 隻，每次隨機召喚不同角色
  • 視覺區分：頭上顯示愛心/星星標記
  • 動畫：跟隨跑步+攻擊（2 組，共用基礎動畫微調）

演出裝飾角色

  • 用途：場景背景觀眾（看台歡呼）、待機畫面演出、回合過場裝飾（舉牌/跑過畫面）
  • 視覺：改貼圖為觀眾風格（加圍巾/旗幟/爆米花等）
  • 動畫：歡呼循環、舉牌（1~2 組）
