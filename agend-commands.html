⌨️ AgEnD 指令手冊

agend-terminal 常用指令與 TUI 操作

🚀 啟動與停止

agend-terminal start
啟動 daemon，載入 fleet.yaml，建立所有 instance
agend-terminal stop
停止 daemon，所有 agent 停止運行
agend-terminal app
啟動 TUI 介面（多分頁管理），可即時查看各 agent 狀態

📋 查看狀態

agend-terminal list
列出所有運行中的 agent（名稱、狀態）
agend-terminal status
詳細狀態（健康狀態、agent state、最後心跳）
agend-terminal doctor
健康檢查（診斷問題）

🔧 操作 Agent

agend-terminal attach <name>
連接到指定 agent 的終端，即時查看輸出。用 Ctrl+B d 離開
agend-terminal inject <name> "訊息"
發送輸入到指定 agent 的 PTY
agend-terminal kill <name>
終止特定 agent

🗂️ Fleet 管理

agend-terminal fleet
Fleet 管理相關子指令

🖥️ TUI 介面操作

啟動方式：agend-terminal app

功能

  • 多分頁顯示 — 每個 agent 一個分頁
  • 即時輸出 — 看到 agent 的思考、執行、回覆過程
  • 分割視窗 — 同時查看多個 agent
  • 直接互動 — 在 TUI 中對 agent 輸入指令

快捷鍵

  • Ctrl+B 然後按數字 — 切換分頁
  • Ctrl+B d — 離開 TUI（daemon 繼續運行）
  • Ctrl+B c — 新建分頁

📌 日常使用

老爺日常只需要：

  • agend-terminal start — 開機啟動（已設定自動啟動）
  • 透過 Telegram 操作即可
  • 需要除錯時才用 agend-terminal app 查看

🔄 其他指令

agend-terminal --version
顯示版本號
agend-terminal quickstart
互動式設定（偵測後端、設定 Telegram、產生 fleet.yaml）
agend-terminal bugreport
產生 bug 報告（含診斷、日誌、設定）
agend-terminal completions powershell
產生 PowerShell 自動補全
